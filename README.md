# Little Secret World

A personal interactive web experience built with React, TypeScript, Vite, Tailwind CSS and shadcn-ui components.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn-ui

## Running Locally

```bash
npm install
npm run dev
```

The app will be available on the local development port shown in your terminal (typically http://localhost:5173).

## Tech Stack

- Vite for fast build and dev server
- React 18 with TypeScript
- Tailwind CSS for styling
- shadcn-ui for pre-built components
